🎬 Netflix Clone Website

A simple Netflix-inspired web application built using HTML, CSS, and JavaScript. This project replicates the UI of Netflix and demonstrates front-end development skills like responsive design, layout structuring, and basic interactivity.

---
 🚀 Features

* 🎥 Netflix-style homepage UI
* 📱 Fully responsive design (mobile + desktop)
* 🎞️ Movie/TV show sections (Trending, Popular, etc.)
* 🔍 Navigation bar with smooth scrolling
* 🎨 Attractive hover effects and animations

---

🛠️ Tech Stack

* **HTML5** – Structure of the website
* **CSS3** – Styling and layout (Flexbox / Grid)
* **JavaScript** – Basic interactivity
